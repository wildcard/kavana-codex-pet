# Attribution

## Kavana

Kavana (כוונה, “intention”) is a custom digital pet inspired by Karo, a real black-and-tan Shiba Inu. The public repository contains only curated derived pet assets: the Codex v2 spritesheet, manifest, source frames, row strips, preview media, contact sheet, and sanitized metadata.

Private photos, raw references, prompts, generation logs, Codex state, and machine-specific files are not included.
