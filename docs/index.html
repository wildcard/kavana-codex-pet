From Karo to Kavana

A real dog, reimagined for the terminal.

Kavana began as Karo, a much-loved Shiba Inu, and was hatched into a full v2 Codex pet. Her name means intention—a small reminder that careful tools and delightful software can belong together.

Install

Bring her home.

Clone the repository, run the installer, then select Kavana under Codex Desktop’s Appearance settings.

git clone https://github.com/wildcard/kavana-codex-pet.git
cd kavana-codex-pet
./scripts/install.sh
