# Kavana Codex Pet v1.0.0

Kavana’s first standalone release packages the Project Caro community companion as a reusable v2 Codex pet.

## Highlights

- Black-and-tan Shiba Inu artwork grounded in Karo’s identity.
- Full 8×11, `spriteVersionNumber: 2` atlas.
- Nine standard Codex state animations plus sixteen clockwise look directions.
- Ready-to-install desktop package and Windows mobile-sync selection helpers.
- Source frames, row strips, contact sheet, ten MP4 previews, metadata, and checksums.
- Cross-platform deterministic verifier and privacy scan.
- Interactive GitHub Pages landing page and transparent Project Caro attribution.

## Install

```bash
git clone https://github.com/wildcard/kavana-codex-pet.git
cd kavana-codex-pet
./scripts/install.sh
```

Windows PowerShell users can run `.\scripts\install.ps1 -Select`.

## Verification

```bash
python3 scripts/verify.py
```
